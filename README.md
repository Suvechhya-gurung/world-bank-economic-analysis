#  Global Economic Trends Analysis (2004–2023)

## Overview
This project analyzes global economic indicators using the World Bank API (via wbgapi) over a 20-year period (2004–2023). The analysis focuses on GDP per capita (PPP), GDP growth, education expenditure, and unemployment rates across countries and regions.

The objective is to identify economic growth patterns, regional disparities, and relationships between education spending and labor market outcomes.

---

## Tools & Technologies
- Python
- wbgapi (World Bank API)
- Pandas
- NumPy
- Jupyter Notebook

---

## Key Analyses

### 1. GDP per Capita by Region
- Aggregated country-level data into regional averages
- Compared long-term economic performance trends

### 2. Fastest & Slowest Growing Economies
- Calculated average GDP growth rates (2004–2023)
- Ranked top and bottom performing countries

### 3. Education Spending vs Unemployment
- Merged education expenditure and unemployment data
- Created scatter plots to analyze macroeconomic relationships

### 4. Global & Regional Unemployment Trends
- Examined changes in labor market performance across regions
